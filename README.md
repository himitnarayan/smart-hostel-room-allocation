# Smart Hostel Room Allocation System

## Tech Stack
- Python
- Django
- PostgreSQL
- Bootstrap
- Render Deployment

## Features
- Add Room
- View All Rooms
- Search Rooms
- Smart Allocation (Smallest Suitable Room)
- Occupancy Tracking
- Full Room Detection
- Allocation History

## How to Run Locally
1. Create virtual environment
2. Install requirements
3. Configure PostgreSQL
4. Run migrations
5. python manage.py runserver
